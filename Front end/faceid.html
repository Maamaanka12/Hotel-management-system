How It Works

  1. 1 Select a customer from the dropdown on the left.
  2. 2 Ask the guest to face the camera directly and click Start Face Scan.
  3. 3 The system scans for ~3 seconds. On success, the customer's Face ID status is saved.
  4. 4 The customer can now use Face ID for future check-ins instead of a key card.

In the full implementation, this connects to a Python + OpenCV backend via a REST API endpoint that performs real facial feature extraction and matching.
