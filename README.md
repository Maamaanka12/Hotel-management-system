# Hotel-management-system
This project is a hotel management system that helps staff manage guests, rooms, bookings, payments, smart room controls, and face-based check-in/enrollment from one place.

What it does
Think of it as the hotel’s digital front desk and operations desk combined. It keeps track of who is staying at the hotel, which rooms are free or booked, what payments have been made, and which room devices are on or off. It also supports face enrollment so a guest’s face can be stored for future biometric use.

How it works
The system is split into two main parts: the frontend and the backend. The frontend is the part people click and use in the browser, while the backend is the engine behind it that talks to the database and performs actions. When a staff member saves a guest, creates a booking, or turns a room light on, the frontend sends a request to the backend, and the backend updates the hotel database.
